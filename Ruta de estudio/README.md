## Markdown: Documentación y Toma de notas

1. Introducción a Markdown:
   
   - Aprender la sintaxis básica de Markdown.
   - Familiarizarse con la estructura de los archivos Markdown.

2. Herramientas de Markdown:
   
   - Explorar las herramientas y editores de Markdown disponibles.
   - Elegir una herramienta adecuada para tomar notas y crear documentación.

3. Uso de Markdown:
   
   - Practicar la creación de documentos utilizando Markdown.
   - Aprender a dar formato a texto, crear listas, agregar enlaces, etc.

4. Exportación de documentos:
   
   - Conocer las opciones de exportación de documentos Markdown.
   - Exportar documentos Markdown a diferentes formatos, como PDF o HTML.

## Git y Github: Trabajo coolaborativo con código

1. Introducción a Git:
   
   - Comprender los conceptos básicos de control de versiones.
   - Aprender los comandos fundamentales de Git, como init, add, commit y push.

2. Configuración de Github:
   
   - Crear una cuenta en Github.
   - Configurar la conexión entre Git local y Github remoto.

3. Repositorios y ramas:
   
   - Crear repositorios en Github.
   - Trabajar con ramas para organizar el desarrollo de proyectos.

4. Colaboración en Github:
   
   - Aprender a clonar repositorios existentes.
   - Contribuir a proyectos de código abierto mediante pull requests.

## Lógica de la programación

1. Fundamentos de la lógica de programación:
   
   - Comprender los conceptos básicos de la lógica de programación.
   - Aprender a descomponer problemas en pasos lógicos.

2. Estructuras de control:
   
   - Familiarizarse con las estructuras de control, como condicionales y bucles.
   - Practicar la aplicación de estas estructuras en programas sencillos.

3. Funciones y algoritmos:
   
   - Aprender a diseñar y utilizar funciones en la programación.
   - Desarrollar algoritmos para resolver problemas específicos.

## Wordpress: El CMS más popular del mercado y un constructor de sitios imparable

1. Introducción a Wordpress:
   
   - Familiarizarse con el entorno de Wordpress.
   - Comprender los conceptos básicos de gestión de contenido con Wordpress.

2. Configuración y personalización:
   
   - Instalar y configurar un sitio web de Wordpress.
   - Personalizar la apariencia utilizando temas y plugins.

3. Creación de contenido:
   
   - Aprender a crear y administrar páginas, entradas y categorías en Wordpress.
   - Utilizar el editor de Wordpress para diseñar y formatear contenido.

4. Personalización avanzada:
   
   - Explorar opciones de personalización más avanzadas, como la creación de plantillas personalizadas.
   - Aprender a utilizar funciones y estilos personalizados en Wordpress.

## Diseño U/X de páginas web

1. Introducción al diseño U/X:
   
   - Comprender los conceptos básicos de diseño de experiencia de usuario (U/X).
   - Conocer la importancia del diseño centrado en el usuario en el desarrollo de páginas web.

2. Principios de diseño U/X:
   
   - Aprender los principios fundamentales de diseño U/X, como la usabilidad y accesibilidad.
   - Familiarizarse con la teoría del diseño visual y la composición de elementos.

3. Investigación y análisis de usuarios:
   
   - Realizar investigaciones de usuarios para comprender sus necesidades y comportamientos.
   - Utilizar técnicas de análisis, como mapas de empatía y diagramas de flujo, para definir perfiles de usuarios y escenarios.

4. Diseño de wireframes y mockups:
   
   - Crear wireframes para representar la estructura y disposición de los elementos en una página web.
   - Utilizar herramientas de diseño, como Adobe XD o Sketch, para crear mockups interactivos y visualizar el flujo de la página.

5. Prototipado y pruebas de usabilidad:
   
   - Desarrollar prototipos interactivos para evaluar la usabilidad de una página web.
   - Realizar pruebas de usabilidad con usuarios reales y recopilar retroalimentación para iterar en el diseño.

6. Diseño visual y branding:
   
   - Aplicar principios de diseño visual, como el uso de colores, tipografías y elementos gráficos.
   - Asegurar que el diseño refleje la identidad de la marca y transmita el mensaje deseado.

## HTML

1. Fundamentos de HTML:
   
   - Aprender los conceptos básicos de HTML, como etiquetas, elementos y atributos.
   - Crear estructuras básicas de una página web utilizando HTML.

2. Elementos y atributos avanzados:
   
   - Explorar elementos HTML más avanzados, como formularios, tablas y multimedia.
   - Utilizar atributos para controlar el comportamiento y estilo de los elementos.

3. Semántica HTML:
   
   - Comprender la importancia de la semántica en HTML.
   - Utilizar elementos semánticos para mejorar la accesibilidad y el SEO.

4. Prácticas recomendadas:
   
   - Aprender buenas prácticas de codificación en HTML.
   - Optimizar la estructura y legibilidad del código HTML.

## CSS

1. Introducción a CSS:
   
   - Comprender los conceptos básicos de CSS.
   - Aprender a aplicar estilos básicos a elementos HTML.

2. Selectores y propiedades:
   
   - Explorar diferentes tipos de selectores en CSS.
   - Utilizar propiedades CSS para modificar el estilo de los elementos.

3. Diseño responsive:
   
   - Aprender a crear diseños responsive utilizando CSS.
   - Utilizar media queries para adaptar el estilo a diferentes dispositivos.

4. Animaciones y efectos:
   
   - Explorar técnicas para crear animaciones y efectos visuales utilizando CSS.
   - Aprender a utilizar transiciones y transformaciones CSS.

## JavaScript

1. Fundamentos de JavaScript:
   
   - Aprender los conceptos básicos de JavaScript, como variables, tipos de datos y operadores.
   - Familiarizarse con la sintaxis y estructura de un programa en JavaScript.

2. Manipulación del DOM:
   
   - Comprender el modelo de objetos del documento (DOM).
   - Utilizar JavaScript para manipular elementos HTML y responder a eventos.

3. Funciones y objetos:
   
   - Aprender a diseñar y utilizar funciones en JavaScript.
   - Trabajar con objetos y métodos para estructurar y organizar el código.

4. Interacción con APIs:
   
   - Explorar cómo interactuar con APIs externas utilizando JavaScript.
   - Aprender a realizar peticiones AJAX y procesar respuestas en formato JSON.

## Inglés aplicado a programación

1. Vocabulario técnico:
   
   - Familiarizarse con el vocabulario técnico utilizado en programación.
   - Aprender los términos comunes relacionados con variables, funciones, estructuras de control, etc.

2. Documentación en inglés:
   
   - Practicar la lectura de documentación técnica en inglés.
   - Aprender a comprender y utilizar recursos como la documentación de bibliotecas y frameworks.

3. Comunicación en inglés:
   
   - Mejorar las habilidades de comunicación en inglés relacionadas con la programación.
   - Practicar la escritura de correos electrónicos, mensajes de chat y comentarios de código en inglés.

4. Tutoriales y cursos en inglés:
   
   - Explorar tutoriales y cursos en línea sobre programación en inglés.
   - Aprovechar los recursos educativos en inglés para ampliar el conocimiento y la comprensión de los conceptos de programación.

5. Participación en comunidades en línea:
   
   - Unirse a comunidades en línea donde se discuten temas de programación en inglés.
   - Participar activamente en foros, grupos de discusión y comunidades de desarrollo para practicar el uso del inglés en un contexto técnico.

6. Lectura de blogs y artículos en inglés:
   
   - Leer blogs y artículos relacionados con la programación en inglés.
   - Mantenerse actualizado sobre las últimas tendencias y novedades en el campo de la programación mediante la lectura de recursos en inglés.
